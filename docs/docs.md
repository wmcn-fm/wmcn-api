##Users

###`GET /`:
-	**URL params:** none
-	**Success response:**
	-	Code: 200
	-	Content: `{}`	//	JSON object  containg every active user of the site (accounts)
	
###`GET /:id`:
-	**URL params:** user's Mongo OID number (*required*)
-	**Success response:**
	-	Code: 200
	-	Content: `{firstName: "Will", lastName: "Kent-Daggett", ...}`
